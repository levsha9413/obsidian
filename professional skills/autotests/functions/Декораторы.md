Декараторы - это функции обертки чтобы усовершенствовать какую-то функцию не изменяя ее.
Пример:

def make_cool(foo):
	def wrapper():
		return str(foo()+'важная инфа')
	return wrapper

def say_hello():
	return 'hello'


чтобы вывести hello важная инфа можно:

def say_hello():
	return 'hello'
say_hello = make_cool(say_hello)


а можно
@make_cool
def say_hello():
	return 'hello'
