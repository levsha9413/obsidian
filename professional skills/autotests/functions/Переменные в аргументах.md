Нельзя инициализировать переменную в аргументе, т.к. она создастся один раз и при повторном вызове функции уже будет не пустая, т.о. функция каждый раз будет выдавать разное поведение.

Неправильно:
def foo(lst, r=[]):
	*делаем что-то с r*
	
Правильно:
def foo(lst, r=None):
	r=[]
	*делаем что-то с r*
